Farming Assistant 

ESTADO: Beta

HOW TO RUN:

Prerequisitos:

    - Tener Reactjs (v6)

    - Tener Nodejs 

    - Tener las extensiones de React para el routing

    - Tener las extensiones de React para bootstrap e iconos de bootstrap.


Run:

En el directorio de app-FA ejecutar

    - npm start

En el directorio de server ejecutar

    - node app.js
